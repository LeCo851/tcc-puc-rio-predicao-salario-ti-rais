Sobre o Projeto

Trabalho de Conclusão de Curso (TCC) desenvolvido para o MBA em Data Science & Big Data da PUC-Rio.

O Objetivo

Este projeto visa resolver a assimetria de informações no mercado de trabalho de TI, fornecendo uma ferramenta capaz de estimar salários com base em dados reais e históricos.

Utilizando a base de dados da RAIS (Relação Anual de Informações Sociais), treinamos um modelo de Machine Learning (LightGBM) capaz de entender padrões complexos entre cargo, escolaridade, localização, raça, idade e porte da empresa para cargos de TI e relacionados.
